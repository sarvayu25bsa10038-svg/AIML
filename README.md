This is a simple, console-based Inventory Management and Billing System built in Python for small retail shops.
Key Features:

Add, Update, Delete, and Search items (name, quantity, price)
Permanent data storage using JSON file (inventory.json)
Full billing system with:
Multiple item purchase
Stock availability check
Automatic stock deduction
18% GST calculation
Professional printed bill (subtotal, GST, final total)

Clean menu with 6 options:
Add Item
Update Item
Delete Item
Search Item
Billing
Exit

User-friendly with clear screen (cls), delays, and error messages
No external dependencies beyond standard Python (json, os, time)

Purpose:
Helps small shop owners manage stock and generate accurate bills quickly without needing internet or complex software.
Status: Fully working, tested, and ready to use.
